🔐 Tor Proxy Setup

To hide your IP address:

1. Install and run Tor Browser or Tor service

2. Ensure Tor SOCKS5 proxy is running on 127.0.0.1:9150

3. Check the "Use Tor Proxy" checkbox before starting

4. Your requests will be routed through Tor network (IP will be hidden)

Without Tor enabled, your real IP address will be visible to the target.
